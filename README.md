# Typer.js

**Typer.js** یک کتابخانه جاوااسکریپت حرفه‌ای برای ایجاد افکت تایپ‌گر در محیط ترمینال یا شبیه ترمینال است. مناسب برای نمایش مهارت‌های برنامه‌نویسی، مفاهیم شی‌گرا و انیمیشن‌های زمان‌بندی شده به صورت تعاملی و زیبا.

## ویژگی‌ها
- شبیه‌سازی **افکت تایپ کردن** در ترمینال یا کنسول  
- استفاده به صورت **شی‌گرا در جاوااسکریپت**  
- قابلیت تنظیم **سرعت و زمان‌بندی تایپ**  
- سبک و آسان برای استفاده و ادغام در پروژه‌ها
